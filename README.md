## Установка всех пакетов
>npm i
## Запуск проекта
>npm start
## Сформировать для разработки
>npm run build:dev
## Сформировать для продукта
>npm run build:prod
____
### Возможности:
- Передвижение по сайту без перезагрузок
- Поиск по сайту
- Возможность передать ссылку на найденые материалы
